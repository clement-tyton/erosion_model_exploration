Priority picks Your context ResNet/NeXt EfficientNet MiT (Segformer) SENet DenseNet MobileOne Others

Your setup constraints

  • 6-band input → must modify first conv layer (all encoders require in_channels=6)
  • Erosion detection → fine-grained boundary accuracy matters → avoid too much downsampling
  • Small target class (erosion rare) → encoder richness and skip connections matter a lot for UNet
  • Remote sensing imagery → advprop / noisy-student weights may help (high dynamic range)
  • For Segformer: MiT encoders are native backbone — other families need adapter layers

Recommended experiment sequence

  • Run 1: UNet + resnet50 (imagenet) → establish baseline F1
  • Run 2: Segformer + mit_b2 (imagenet) → see if transformer head beats UNet baseline
  • Run 3: UNet + efficientnet-b4 (advprop) → best CNN challenger
  • Run 4: UNet + se_resnext50_32x4d → channel attention variant
  • Run 5: Segformer + mit_b3 → scale up transformer if run 2 was best
  • Run 6+: noisy-student variants, densenet161, mit_b4 based on gap analysis
